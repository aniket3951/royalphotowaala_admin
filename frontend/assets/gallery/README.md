# Gallery Images Directory

This directory contains images for the website gallery.

## Organization:
Images should be organized by category:
- `wedding/` - Wedding photography
- `portrait/` - Portrait sessions
- `events/` - Event photography
- `candid/` - Candid shots

## File Naming:
Use descriptive names like:
- `wedding-couple-001.jpg`
- `portrait-family-002.jpg`
- `event-birthday-003.jpg`

## Guidelines:
- Optimize images for web (max 1MB per image)
- Use JPG format for photos
- Maintain high quality but reasonable file size
- Consider responsive image versions
- Include alt text descriptions in the CMS
