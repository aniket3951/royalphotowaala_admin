# Home Images Directory

This directory contains images for the homepage slideshow.

## Purpose:
These images are displayed as background slides on the homepage hero section.

## Guidelines:
- Recommended size: 1920x1080px (16:9 aspect ratio)
- Use high-quality, impactful images
- Optimize for web loading (max 2MB per image)
- JPG format recommended
- 5-10 images ideal for slideshow rotation
- Images should showcase your best work

## File Naming:
Use sequential naming:
- `slide-001.jpg`
- `slide-002.jpg`
- `slide-003.jpg`

## Tips:
- Choose diverse images showing different photography styles
- Ensure good lighting and composition
- Consider text readability when overlayed
- Update regularly to keep content fresh
